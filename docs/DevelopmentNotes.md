# Notes
## Build

```
npm i
```
After changing svg-editor.js and possibly other js files, run this to minify. Output is in dist/.
```
npm run rollup
```
After changing svg-editor-es.html, run this, output is svg-editor.html.
```
npm run build-html
```
Run this to build everything
```
npm run test-prep
```
## Serve
```
npm run start
```
